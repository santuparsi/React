# How to setup react

There are 2 ways we can setup react app

npm - node package manager
npx - node package runner


> npm install create-react-app -g
> create-react-app <project_name>
> cd <project_name>
> npm start

or

> npx create-react-app <project_name>
> cd <project_name>
> npm start
